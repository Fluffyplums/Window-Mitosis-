# Window-Mitosis-
A cool little game i made that uses 2 coding engines!

--Window Mitosis---

I just made this for no apparent reason lol
------------------------------
*Disclaimer: This project does not follow the meaning of "mitosis" at all. (more like Window Aphids)
It is only a project made out of boredom and sometimes passion.
You could even call it the most boring game on the planet.

Starts off as a simple window, saying a number. However if you close it, it will open again. If you try to close it more than 10 times, then it turns into a "section". Once you have cleared all 4 sections, you will end up with 4 windows. All windows will have the same properties, hence (a part of) the meaning of "mitosis" which is the multiplication of bacteria. It will multiply into 16, 32, 64, and so on.

*Note: Sometimes your firewall may see any of the files as potentially dangerous, since it would seem they have no author. 
If you notice any errors with any Batch or Visual Basic file that may seem safe to your eyes:

* And allow the file.

If you press "Cancel", then the file won`t be able to load.
To stop the "malware", close or press Yes on the Quit Box that accompanies you at the start of the game.
